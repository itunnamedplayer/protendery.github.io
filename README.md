# ПроТендеры — инструкция по деплою на Vercel

## Структура проекта
```
protendery/
├── api/
│   └── send.js          ← серверная функция отправки в Telegram (токен скрыт)
├── docs/                ← папка для фото документов/сертификатов
│   ├── cert1.jpg        ← сертификат ЦСК
│   ├── cert2.jpg        ← сертификат Сбербанк-АСТ
│   ├── cert3.jpg        ← удостоверение МГТУ Баумана
│   ├── doc4.jpg  ...    ← остальные документы (до doc13.jpg)
├── index.html           ← главная страница
├── vercel.json          ← конфигурация Vercel
└── README.md
```

## Шаг 1 — Добавьте фото документов
Поместите 13 фотографий сертификатов в папку `docs/`:
- `cert1.jpg`, `cert2.jpg`, `cert3.jpg` — уже настроены в коде
- `doc4.jpg` ... `doc13.jpg` — замените подписи в массиве `docPhotos` в `index.html`

Рекомендуемый формат: JPG, ширина 800–1200px, сжатые (TinyJPG.com).

## Шаг 2 — Создайте Telegram-бота
1. Напишите @BotFather → `/newbot` → придумайте имя → получите токен вида `7123456789:AAFxxxx`
2. Напишите вашему боту любое сообщение
3. Узнайте свой chat_id: откройте `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   - Найдите `"chat":{"id": ЧИСЛО}` — это и есть ваш chat_id

## Шаг 3 — Деплой на Vercel
1. Зарегистрируйтесь на vercel.com
2. Установите Vercel CLI: `npm i -g vercel`
3. В папке проекта выполните: `vercel`
4. Следуйте инструкциям (выберите "новый проект")

**ИЛИ через GitHub:**
1. Залейте папку в GitHub-репозиторий
2. Зайдите на vercel.com → "New Project" → импортируйте репозиторий
3. Нажмите Deploy

## Шаг 4 — Добавьте переменные окружения
В Vercel Dashboard → ваш проект → Settings → Environment Variables:

| Имя переменной | Значение |
|---|---|
| `TG_BOT_TOKEN` | `7123456789:AAFxxxxxxxxxxxxxxx` |
| `TG_CHAT_ID`   | `123456789` |

После добавления переменных нажмите **Redeploy**.

## Шаг 5 — Проверка
Откройте сайт, заполните форму — в течение нескольких секунд придёт сообщение в Telegram.

## Добавление нового контакта в поля
В `index.html` найдите и замените:
- `+7 (XXX) XXX-XX-XX` → ваш номер телефона
- `https://t.me/ВАША_ССЫЛКА` → ссылка на ваш Telegram (например `https://t.me/protendery`)
- `© 2026 ПроТендеры` → при необходимости измените год
